# 游戏兑换码查找

搜索并整理最新的游戏兑换码/礼包码/CDK，优先从B站获取最新发布的信息。

## 何时使用

- 用户询问任何游戏的兑换码、礼包码、CDK、激活码
- 用户想知道某个游戏有什么福利/奖励可以领取
- 用户说"XX游戏有什么兑换码"、"XX游戏最新礼包"

## 步骤

### 1. B站优先搜索
先用 `bilibili_search` 搜索关键词 `"{游戏名} 兑换码"` 或 `"{游戏名} 礼包码"`。
B站UP主通常会第一时间发布最新兑换码，评论中有时效性验证。

### 2. 网页搜索补充
如果B站结果不足（少于3个有效兑换码），用 `web_search` 搜索 `"{游戏名} 兑换码 最新 {当前年份}"`。

### 3. 详情验证
对搜索到的链接用 `web_fetch` 抓取具体内容，确认：
- 兑换码格式是否正确
- 有效期是否还有效
- 是否有使用次数限制

### 4. 整理输出
按以下格式返回：

```
🎮 {游戏名} 兑换码汇总

| 兑换码 | 奖励内容 | 有效期 | 来源 |
|--------|---------|--------|------|
| XXX-XXX | 钻石x100 | 7月30日 | B站UP主XXX |

⚠️ 注意: 兑换码可能有次数限制，请尽快使用。
```

## 输出要求

- 标注每个兑换码的来源链接
- 标注发布/过期时间
- 优先显示B站来源（通常最新）
- 提醒用户尽快兑换

## 技巧

1. B站视频评论区和动态区常有玩家分享新码，比搜索引擎更快
2. 搜索时强制加上当前年份避免过期码
3. 有多个游戏时，分别搜索各自兑换码
4. 如果兑换码格式异常（如中文混入），可能是OCR提取错误，需核实原文
