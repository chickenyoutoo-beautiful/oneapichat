# 游戏兑换码查找

搜索并整理最新的游戏兑换码/礼包码/CDK，优先从B站获取最新发布的信息。

**★ B站是兑换码第一手来源** — UP主动态和评论区比任何搜索引擎都快。

## 何时使用

- 用户询问任何游戏的兑换码、礼包码、CDK、激活码
- 用户想知道某个游戏有什么福利/奖励可以领取
- 用户说"XX游戏有什么兑换码"、"XX游戏最新礼包"

## 步骤

### 1. B站优先搜索（三路并行）
先用 `bilibili_search` 搜索:
- `"{游戏名} 兑换码"` — 视频标题+简介
- `"{游戏名} 礼包码"` — 专栏文章常有汇总列表
- 同时找相关UP主的 `bilibili_dynamic_list` — 动态比视频发布更快

B站UP主通常会第一时间发布最新兑换码，评论区也有玩家分享和时效性验证。

### 2. 专栏提取
对搜索到的专栏文章 → `bilibili_article_read(cvid)` 获取完整兑换码列表。

### 3. 网页搜索补充
如果B站结果不足（少于3个有效兑换码），用 `web_search` 搜索 `"{游戏名} 兑换码 最新 {当前年份}"`。

### 4. 详情验证
对搜索到的链接用 `web_fetch` 抓取具体内容，确认：
- 兑换码格式是否正确
- 有效期是否还有效
- 是否有使用次数限制

### 5. 整理输出
按以下格式返回：

```
🎮 {游戏名} 兑换码汇总

| 兑换码 | 奖励内容 | 有效期 | 来源 |
|--------|---------|--------|------|
| XXX-XXX | 钻石x100 | 7月30日 | B站UP主XXX |

⚠️ 注意: 兑换码可能有次数限制，请尽快使用。
来源链接: {url1}, {url2}
```

## 输出要求

- 标注每个兑换码的来源链接
- 标注发布/过期时间
- 优先显示B站来源（通常最新）
- 提醒用户尽快兑换

## 技巧

1. B站视频评论区和动态区常有玩家分享新码，比搜索引擎更快
2. UP主动态比出视频更快 — 先发动态预告再出视频
3. 搜索时强制加上当前年份避免过期码
4. 有多个游戏时，分别搜索各自兑换码
5. 如果兑换码格式异常（如中文混入），可能是OCR提取错误，需核实原文
6. 专栏文章常有定期更新的兑换码汇总系列 — 关注该UP主看历史
