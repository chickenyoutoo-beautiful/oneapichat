# Xiaomi MiMo Orbit 100T Token 激励计划 FAQ

## 01 什么是 Xiaomi MiMo Orbit 百万亿 Token 创造者激励计划？
Xiaomi MiMo Orbit 百万亿 Token 创造者激励计划是 Xiaomi MiMo 面向全球高质量、AI 驱动的用户的限时 Token 定向发放活动。计划在 30 天内发放总计 100 万亿 Token 权益，完全免费。每一份申请都会经过仔细评估。
Xiaomi MiMo(https://mimo.xiaomi.com/) 是小米自研的大模型，V2.5 系列包含旗舰推理模型、多模态模型和语音合成模型。
MiMo API 开放平台(http://platform.xiaomimimo.com) 面向全球用户提供 API 推理服务，可直接对接 Claude Code、Cursor、OpenClaw 等编程工具。

## 02 活动时间和参与资格？
北京时间 2026 年 4 月 28 日 00:00 至 5 月 28 日 00:00(UTC 2026 年 4 月 27 日 16:00 至 5 月 27 日 16:00)，共 30 天，赠完即止。面向全球所有用户开放，个人开发者、团队、企业均可参与。

## 03 如何加入该项目？
1. 在活动页面点击「立即申请」，填写并提交表单
2. 等待评估（约 3 个工作日）
3. 评估通过后，你会收到一封通知邮件
4. 使用你填写的邮箱登录或注册 Xiaomi MiMo API 开放平台，权益会在收到邮件后的 24h 内到账

## 04 确保权益顺利发放？
请在提交表单前务必确认以下事项，否则可能导致权益无法到账：
- 已有开放平台账号？请确保它已由你在表单中填写的邮箱注册。如果它由手机号注册的，请绑定该邮箱。
- 还没有开放平台账号？请在收到评估通过的邮件后，根据邮件的提示，使用你填写的邮箱完成账号注册。
- 绑定邮箱地址：https://account.xiaomi.com/fe/service/account

## 05 我会获得什么权益？
根据表单填写内容以及开放平台账号状态，发放不同额度、不同档位的 Token Plan，或不同金额的赠金，完全免费。
Token Plan 是开放平台的订阅制套餐，可直接配置到 Claude Code、Cursor 等编程工具中使用 MiMo 系列模型。赠金将直接充入开放平台账户余额。

## 06 评估标准是什么？
根据提交的表单内容进行综合评估，包括使用的 AI 工具、底层模型、项目描述和证明材料。填写越详细、项目越具体，评估通过率和权益档位越高。
如果 3 天内没有收到评估通过的邮件，可以重新提交表单。

## 07 权益有有效期吗？
有。不同权益的有效期以到账提示为准，过期未使用的权益将自动失效。

## 08 没收到邮件怎么办？
1. 检查垃圾邮件箱
2. 确认表单中填写的邮箱地址是否正确
3. 若 3 个工作日内仍未收到，尝试重新申请
4. 如仍未收到，通过开放平台「联系我们」(https://platform.xiaomimimo.com/contact) 反馈

## 09 邮箱不一致怎么办？
- 使用「申请用邮箱」注册新的小米账号，登录开放平台即可接收权益
- 将原有小米账号的邮箱换绑为「申请用邮箱」，更换地址：https://id.mi.com
- 权益会在 24 小时内自动到账
